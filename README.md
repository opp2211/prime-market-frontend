# prime-market-frontend

Фронтенд Prime Market. Одностраничное приложение на React с Vite, маршрутизацией и запросами к API.

Prime Market - сайт-биржа игровых ценностей, где пользователи могут купить и продать товары или услуги, а также
выставить свое предложение на продажу или покупку.

## Стек
- React 19 + Vite
- React Router
- Axios
- ESLint
- Node.js + npm
